# RJF - NGO/Foundation Management System

This is a complete management system for non-governmental organizations or foundations. It contains various modules for managing members, events, and organizations effectively.

## Project Structure

- `src/`: Contains the main source code.
- `docs/`: Documentation files.
- `tests/`: Unit and integration tests.

## Setup

To set up the project, ensure you have Python installed along with the required packages listed in `requirements.txt`.

## Usage

To run the application:

```bash
python src/main.py
```
